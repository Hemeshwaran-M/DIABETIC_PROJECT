# 🩺 Diabetes Prediction System using XGBoost

A Machine Learning web application that predicts the likelihood of diabetes based on medical parameters. The project uses the XGBoost algorithm with hyperparameter tuning for improved accuracy and is deployed using Streamlit for easy access.

## 📦 Installation

Install all required packages:

```bash
pip install -r requirements.txt
```

---

## 🎯 Train the Model

Run the following command:

```bash
python train_model.py
```

This will:
- Load the diabetes dataset
- Preprocess the data
- Train the XGBoost model
- Perform hyperparameter tuning
- Save the trained model and scaler

---

## ▶️ Run the Web Application

```bash
streamlit run app.py
```

After running the command, Streamlit will launch the application in your default web browser.

---

## 📊 Model Details

The application uses:

- XGBoost Classifier
- GridSearchCV for Hyperparameter Tuning
- StandardScaler for Feature Scaling
